# 💻 Laptop Price Analysis and Prediction

## 📌 Project Overview

This project focuses on analyzing laptop specifications and predicting laptop prices using Machine Learning.

The goal is to identify key factors influencing laptop prices and build a regression model capable of estimating the price of a laptop based on its specifications.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Laptop Price Prediction

- Premium laptops generally have SSD storage.
- RAM significantly affects laptop pricing.
- Gaming laptops tend to have higher prices due to dedicated GPUs.
